코딩 벤치마크 13% 향상, Claude Opus 4.7이 조용히 출시됐어요

• Anthropic이 Claude Opus 4.7을 정식 출시했어요. 93개 코딩 과제 벤치마크에서 이전 버전 대비 13% 높은 점수를 기록했고, Opus 4.6과 Sonnet 4.6이 둘 다 못 풀었던 문제 4개를 혼자 해결했어요.
• 오래 걸리는 복잡한 작업에서 지시 사항을 더 잘 따르고 스스로 결과를 검증하는 능력이 좋아졌어요. 고해상도 이미지 처리 같은 비전 기능도 함께 강화됐고요. Amazon Bedrock, Google Cloud Vertex AI, Microsoft Foundry 모두에서 바로 쓸 수 있어요.
• 가격은 Opus 4.6과 동일하게 입력 토큰 100만 개당 5달러, 출력 토큰 100만 개당 25달러예요. 사이버 보안 관련 기능은 의도적으로 제한했고, 보안 전문가라면 별도 인증 프로그램에 등록해야 고위험 기능에 접근할 수 있어요.

안드로이드 앱 개발, AI 에이전트랑 같이 하면 진짜 3배 빨라질까요

• Google이 에이전트 기반 개발 흐름에 맞춘 Android CLI와 Android Knowledge Base를 새로 공개했어요. 내부 실험에서 LLM 토큰 사용량을 70% 이상 줄이면서 작업 완료 속도는 3배 빨라졌다고 해요.
• Android Skills는 마크다운 기반 모듈형 명세서예요. 프롬프트 내용에 따라 자동으로 적절한 스킬이 실행되는 구조라, 에이전트가 안드로이드 SDK를 훨씬 효율적으로 다룰 수 있어요. android create 명령어 하나로 권장 아키텍처가 적용된 프로젝트를 바로 만들 수도 있고요.
• CI 파이프라인이나 반복 자동화 작업에도 쓸 수 있어요. Gemini in Android Studio의 에이전트 모드, 플래닝 모드와 함께 쓰면 활성 개발 단계에서도 도움이 돼요. 다만 아직 프리뷰 단계라 인터페이스가 바뀔 수 있어요.

💡 오늘의 인사이트

토큰 70% 절감이라는 숫자가 인상적인데, 이건 단순히 비용 얘기가 아니에요. 에이전트가 불필요한 맥락을 덜 소비한다는 뜻이고, 그만큼 정확한 작업을 더 빨리 한다는 거예요. 안드로이드 개발을 하고 있다면 Android Skills 저장소를 한번 살펴보세요. 팀 내 반복 작업을 스킬로 정의해두면 나중에 꽤 유용하게 쓸 수 있을 거예요.

WAVE 03 57 pts · 14 comments

2년 안식년 끝에 만든 언어, Clojure가 다큐멘터리로 나왔어요

• Clojure는 Rich Hickey가 2년간의 안식년을 보내며 만든 언어예요. JVM 위에서 동작하고, ClojureScript로 자바스크립트 환경에서도 쓸 수 있어요. 세계 최대 핀테크 중 하나인 Nubank가 이 언어를 핵심 스택으로 쓰고 있어요.
• 불변 값과 영속 자료구조를 언어 설계의 중심에 뒀어요. 상태 변화로 생기는 복잡함을 줄이는 게 핵심 철학이에요. 동시성 처리는 STM(Software Transactional Memory)으로 다뤄요.
• Nubank가 2020년에 Clojure를 만든 회사 Cognitect를 인수하면서 언어 개발을 직접 이끌고 있어요. Datomic, Babashka, Jank, ClojureDart 같은 생태계도 함께 소개돼요.

💡 오늘의 인사이트

Clojure를 당장 쓸 일이 없더라도, 이 다큐멘터리는 볼 만해요. 한 언어가 어떤 문제의식에서 태어나고, 어떻게 커뮤니티와 함께 살아남는지를 잘 보여주거든요. 불변성과 단순함에 대한 Rich Hickey의 생각은 어떤 언어로 개발하든 설계에 대해 다시 생각하게 만들어요. 출근길 영상으로 딱이에요.
